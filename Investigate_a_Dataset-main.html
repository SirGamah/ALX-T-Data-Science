INTRODUCTION¶

Dataset Description¶

This data set contains information about 10,000 movies collected from The Movie Database (TMDb), including user ratings and revenue.

Description of Columns

  • 1. id - Index ID of Movies
  • 2. imdb_id - IMBD Movie ID
  • 3. popularity - Popularity of the movie
  • 4. budget - Budget of the Movie
  • 5. revenue - Revenue generated from the movie
  • 6. original_title - Movie title
  • 7. cast - Mames of the Movie casts
  • 8. homepage - URL to the Movie
  • 9. director - Name the Movie Director
  • 10. tagline - Movie's Tagline
  • 11. keywords - Keywords from the Movie's Tagline
  • 12. overview - A sentence summary of the Movie
  • 13. runtime - Movie Duration
  • 14. genres - Genres of the Movie
  • 15. production_companies - Production Companies of the movie
  • 16. release_date - Date on which the movie was released
  • 17. vote_count - Vote count for the movie
  • 18. vote_average - Rating of the Movie
  • 19. release_year - Release year of the movie
  • 20. budget_adj - Adjusted budget due to 2010 inflation
  • 21. revenue_adj - Adjusted revenue due to 2010 inflation

Questions for the Analysis¶

  1. Which directors have the most movies?
  2. Which are the popular genres over the years?
  3. Which movies genereated the highest/lowest profit?
  4. What is the trends of movie runtime over the the decades?

# Defining a 'plot_bar' function.
def plot_bar(x, y, data, title = " ", xlabel = " ", ylabel = " ", rotation = 90):
    plt.figure(figsize = (15, 10))
    bar_list = sns.barplot(x=x, y=y,)
    plt.title(title, fontsize = 20)
    plt.xlabel(xlabel, fontsize = 15)
    plt.ylabel(ylabel, fontsize = 15)
    plt.xticks(rotation = 90)
    
    return plt.show()
